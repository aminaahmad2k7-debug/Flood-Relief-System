Flood Relief Management System (Flask)

This is a Flask-based web application developed to manage flood relief data.
It allows users to upload relief requirement data, process it, and display
results on a dashboard.

## Features
- Upload CSV files
- Process relief requirements
- Display data on dashboard

## Technologies Used
- Python
- Flask
- HTML/CSS
- Pandas

## How to Run
```bash
pip install -r requirements.txt
python app.py
